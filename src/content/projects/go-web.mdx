## Go Web

Go Web is the official corporate website of GoldenOwl company, showcasing the company's products, services, mission, values, and culture. The website serves as a central hub for customers, partners, and potential employees to learn more about the company.

### My Responsibilities

- **Framework Migration:** Migrate the Go Web frontend source code from ReactJS to NextJS.
  - This will involve refactoring components, updating routing, and optimizing for NextJS features like server-side rendering and static site generation.
  - Ensure seamless integration with existing backend systems and data sources.

### Technology Stack

- Backend:
  - [NextJS](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- Frontend:
  - [NextJS](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
  - [React](https://react.dev/) - React is the library for web and native user interfaces. Build user interfaces out of individual pieces called components written in JavaScript.
- Database:
  - [Postgresql](https://www.postgresql.org/) - PostgreSQL also known as Postgres, is a free and open-source relational database management system (RDBMS) emphasizing extensibility and SQL compliance.
