## PAY WITH PACE

PAY WITH PACE is an application that allow customer to split any online purchase into 3 smaller, interest-free payments with Pace Card. With no fees, the Pace Card lets you split the cost of your purchases with more brands.

### My Tasks

- Building marketing website (landing page, campaign page) for Pace company
- Work with many e-commerce platforms (shopify, woocommerce, opencart, magento...) to develop widget/plugin for merchant to integrate with Pace's payment system
- Investigate and resolve merchant issue about integration with Pace

### Stack

- [Gatsby](https://www.gatsbyjs.com/) - Gatsby is a React-based open source framework with performance, scalability and security built-in.
- [React](https://react.dev/) - React is the library for web and native user interfaces. Build user interfaces out of individual pieces called components written in JavaScript.
- [Laravel](https://laravel.com/) - Laravel is the PHP Framework for Web Artisans.
